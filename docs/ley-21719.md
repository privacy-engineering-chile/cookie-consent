# Contexto chileno y Ley 21.719

La Ley 21.719 moderniza el marco chileno de proteccion de datos personales. Este proyecto entiende el consentimiento de cookies como un problema de diseno tecnico, experiencia de usuario y trazabilidad, no como un simple banner generico.

`cookie-consent-cl` no garantiza cumplimiento normativo por si solo y no reemplaza asesoria legal. Su objetivo es ayudar a implementar consentimiento claro, especifico, configurable y auditable desde el frontend.

Principios aplicados:

- Las categorias no esenciales parten desactivadas.
- Rechazar cookies no esenciales debe ser tan visible como aceptar.
- El cierre del banner no equivale a consentimiento.
- La persona puede cambiar sus preferencias despues.
- El lenguaje debe ser comprensible para usuarios en Chile.
