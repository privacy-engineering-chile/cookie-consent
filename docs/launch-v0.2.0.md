# Lanzamiento cookie-consent-cl v0.2.0

## Mensaje central

Lanzamos `cookie-consent-cl v0.2.0`: un gestor open source de consentimiento de cookies para sitios web chilenos, con configurador visual, defaults eticos y soporte para Google Consent Mode v2.

## Post para LinkedIn

Hoy publicamos cookie-consent-cl v0.2.0.

Es un gestor open source de consentimiento de cookies pensado para sitios web chilenos: liviano, sin backend, sin cuentas y sin tracking propio.

Incluye:

- banner y panel de preferencias por categoria
- aceptar, rechazar y configurar con el mismo peso etico
- bloqueo de scripts hasta consentimiento
- Google Consent Mode v2
- configurador visual para PYMEs y equipos tecnicos
- CDN via npm + jsDelivr

No reemplaza asesoria legal ni garantiza cumplimiento por si solo. La idea es ofrecer una base tecnica clara, auditable y alineada con privacidad por diseno.

Repo:
https://github.com/privacy-engineering-chile/cookie-consent

npm:
https://www.npmjs.com/package/cookie-consent-cl

## GitHub Release

Titulo:

```txt

```

Resumen:

```txt
Adds font customization, a redesigned SME-focused configurator, clearer integration feedback, ethical review, auditable summaries, CDN installation via jsDelivr, and improved required-category UX.
```

Highlights:

- `theme.fontFamily` para personalizar tipografia sin cargar recursos externos.
- Configurador visual para PYMEs chilenas.
- Mapa de herramientas para generar snippets bloqueados por consentimiento.
- Revision antes de instalar con buenas practicas sin dark patterns.
- Resumen de configuracion copiable como bitacora tecnica.
- Categoria `Necesarias` como estado fijo, no como toggle clickeable.
- CDN versionado con `cookie-consent-cl@0.2.0`.
