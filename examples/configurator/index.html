Crea tu banner de cookies para Chile

Asistente de configuracion para un banner claro, especifico por categoria y sin dark patterns, creado con cookie-consent-cl.

Modo etico recomendado activado
  • Rechazar no necesarias es tan visible como aceptar.
  • Las categorias opcionales parten desactivadas.
  • El usuario puede revisar preferencias despues.

Contraste suficiente.

Define categorias especificas, sus senales para Google Consent Mode y cookies conocidas. Las categorias opcionales quedan desactivadas por defecto.

Necesarias

Google Consent Mode signals

Para cookies necesarias normalmente se mantiene security_storage concedido.

Analitica

Google Consent Mode signals

Cuando esta categoria se autoriza, las senales marcadas se actualizan como granted.

Tracking e integraciones

Scripts bloqueados por categoria Usa este atributo para cargar integraciones solo despues del consentimiento. <script type="text/plain" data-cookie-consent="analytics" data-src="https://www.googletagmanager.com/gtag/js?id=G-XXXX"></script>
Marketing

Google Consent Mode signals

Para Google Ads, normalmente se usan juntas ad_storage, ad_user_data y ad_personalization.

Tracking e integraciones

Ejemplo: video o pixel publicitario El iframe o script queda sin cargar hasta que la persona acepte marketing. <iframe data-cookie-consent="marketing" data-src="https://www.youtube.com/embed/XXXX"></iframe>
Preferencias

Google Consent Mode signals

Marca solo las senales que realmente correspondan a esta categoria.

Tracking e integraciones

Preferencias del sitio Ideal para scripts que recuerdan idioma, region o ajustes de experiencia.

Texto de preferencias avanzado

Paso 1: agrega CSS, JS y configuracion

Incluye este bloque antes de cerrar el <head> o antes de cerrar el <body>. La configuracion refleja lo seleccionado en este asistente.

Paso 2: instalacion local para desarrollo

Para probar desde este repositorio, instala dependencias, levanta Vite y abre el configurador o los ejemplos.

npm install
npm run dev

Paso 3: scripts bloqueados y Google Consent Mode

Agrega type="text/plain" y data-cookie-consent a integraciones de terceros. Las senales de Google se actualizan segun las categorias que la persona autorice.
