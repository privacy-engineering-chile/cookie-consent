Instrucciones de instalación

Agrega este HTML a tu sitio. La opcion recomendada usa jsDelivr con una version fija: no uses @latest en produccion.

1

Carga CSS y JS

Usa las URLs versionadas de jsDelivr para cargar cookie-consent-cl.css y cookie-consent-cl.iife.js.

2

Pega el HTML

Agrega el bloque generado dentro del <head> de tu página y deja la configuración cerca del cierre de <body> si prefieres separarla.

3

Revisa scripts bloqueados

Si agregaste integraciones, conserva los snippets con type="text/plain" y data-cookie-consent.
